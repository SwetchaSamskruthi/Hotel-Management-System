**Description**
This is a console-based application that simulates a hotel room reservation system. It allows users to select a room type, provide customer details, calculate costs including taxes, and receive a formatted receipt.

**Features**
-Room types supported: Single, Double AC, Double Non-AC, Family AC
-Calculates total charges including GST (18%) and maintenance (10%)
-Randomly assigns a unique room ID (prevents duplicates)
-Generates a booking ID and prints a detailed receipt

**Concepts Used**
-Java arrays and ArrayList
-Random number generation
-Conditionals and loops
-User input and formatted output
